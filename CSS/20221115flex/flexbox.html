flex box

Flex box 設定分為兩部分:彈性容器(flex container)與彈性項目(flex items)。

彈性盒設定

1. display:flex

設定父元素 display:flex為彈性容器，子元素們變成彈性項目(無論是什麼類型的盒子，都會被區塊化)。

彈性容器、彈性項目各自擁有屬性設定

2.flex-direction 彈性盒軸向

彈性盒布局是以主軸盒次軸為基礎，flex-direction可以決定主軸水平或垂直，決定主軸方向後(次軸另一個方向)，知道主、次軸起點與終點，可以做為之後對齊的依據。

由於主軸方向是可變的，才會使用起點(start)、終點(end)而不是左右上或下

預設為flex-direction:row

2.flex-grow彈性伸展係數

彈性項目的伸展係數，決定彈性項目是否伸展，預設0為不伸展

設成1或其他數字，會將剩餘毒空間進行比例分配給彈性項目，例如三欄設定1、2、3，剩餘的空間比例分配為1/6、2/6、3/6
